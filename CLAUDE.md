## 🛠️ 기술 스택

| 항목 | 내용 |
|---|---|
| 프론트엔드 | Vanilla HTML/CSS/JS (빌드 없음) |
| 데이터베이스 | Supabase (PostgreSQL) |
| 배포 | GitHub Pages |
| 폰트 | Google Fonts (Jua, Nanum Gothic) |

### Supabase 테이블 구조

**`quiz_results`** — 퀴즈 정답 기록
```
id, name, phone, question_id, points, created_at
```

**`questions`** — 퀴즈 문제
```
id, q, answer, hint, cat, fb_ok, fb_ng, active, sort_order
```

**`settings`** — 게임 설정값 (key-value)
```
key, value (JSON), updated_at

주요 키:
- game_texts      : 인트로/대시보드/직급 텍스트
- game_config     : 일일 제한, 상품권 기준 pts
- notice          : 공지사항
- roulette_config : 롤플레잉 룰렛 카테고리/항목
```

## 🎮 주요 기능

### 상담 퀴즈
- 50문항 텍스트 입력 퀴즈
- 정답 시 포인트 적립 (1회차 20pts / 2~3회차 10pts / 4회차+ 5pts)
- 오답 시 wrongAnswers 배열에 저장 → 결과 화면에서 복습 가능
- 일일 문제 수 제한 (어드민 설정 가능)
- 스트릭(연속 정답) 시스템

### 직급 시스템
| 직급 | 포인트 범위 |
|---|---|
| 😊 신입 상담사 | 0 ~ 999 pts |
| 📚 열공 상담사 | 1,000 ~ 2,999 pts |
| ☕ 여유 상담사 | 3,000 ~ 5,999 pts |
| 🏅 베테랑 상담사 | 6,000 ~ 9,999 pts |
| 👑 전설의 상담왕 | 10,000 pts+ |

### 롤플레잉 룰렛
- 드럼 슬롯 방식으로 주제 랜덤 선택
- STOP 버튼으로 멈춤
- 카테고리/항목 어드민에서 자유 편집 가능

### 랭킹
- **이달의 TOP 5** (월간, 퀴즈 점수 기준)
- 나의 순위 표시
